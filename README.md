# light-hub

Web-based controller for ELK-BLEDOM-family BLE LED strips (Lotus Lantern, DayBetter, etc.). Runs in [Bluefy](https://apps.apple.com/us/app/bluefy-web-ble-browser/id1492822055) on iOS or Chrome on desktop. No native app, no Apple Developer account, no hub.

Status: in development. See `docs/superpowers/plans/` for the implementation plan.
