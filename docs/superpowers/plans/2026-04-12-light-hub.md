# light-hub Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a vanilla-JS Progressive Web App that controls 3 BLE LED strips (2 Lotus Lantern + 1 DayBetter) over Web Bluetooth from inside the Bluefy iOS browser, replacing the friction of switching between two vendor apps.

**Architecture:** Single-page hash-routed PWA, no build step, ES modules. Two BLE protocols: **ELK-BLEDOM** (Lotus Lantern, service `fff0`/char `fff3`, 9-byte `7E…EF` frames) and **QC Light Protocol** (DayBetter "Smart Light", service `ff10`/char `ff12`, A0-framed with CRC-16/MODBUS). Protocol selected per-device from matched service UUID. Per-device state in localStorage with stable internal `localId`s decoupled from Web Bluetooth's `device.id`. Mock driver for desktop UI dev. Dark ambient visual style.

**Tech Stack:** Vanilla HTML/CSS/JS, native ES modules, Web Bluetooth API, localStorage. No npm, no bundler, no framework. GitHub Pages for hosting.

**Spec:** `docs/superpowers/specs/2026-04-12-light-hub-design.md`

export function assert(condition, message) {
  if (!condition) throw new Error(message || 'Assertion failed');
}

export function assertEqual(actual, expected, message) {
  if (actual !== expected) {
    throw new Error((message || 'assertEqual') + `: expected ${JSON.stringify(expected)}, got ${JSON.stringify(actual)}`);
  }
}

export function assertDeepEqual(actual, expected, message) {
  const a = JSON.stringify(actual);
  const e = JSON.stringify(expected);
  if (a !== e) {
    throw new Error((message || 'assertDeepEqual') + `: expected ${e}, got ${a}`);
  }
}

export function assertBytes(actual, expected, message) {
  const actualArr = Array.from(actual);
  const expectedArr = Array.from(expected);
  if (actualArr.length !== expectedArr.length || actualArr.some((b, i) => b !== expectedArr[i])) {
    const fmt = (arr) => '[' + arr.map(b => '0x' + b.toString(16).padStart(2, '0')).join(', ') + ']';
    throw new Error((message || 'assertBytes') + `: expected ${fmt(expectedArr)}, got ${fmt(actualArr)}`);
  }
}

export async function runSuite(name, tests) {
  const results = [];
  for (const [testName, fn] of Object.entries(tests)) {
    try {
      await fn();
      results.push({ suite: name, test: testName, ok: true });
    } catch (e) {
      results.push({ suite: name, test: testName, ok: false, error: e.message });
    }
  }
  return results;
}

const clampByte = (n) => Math.max(0, Math.min(255, Math.round(n)));
const clampPercent = (n) => Math.max(0, Math.min(100, Math.round(n)));

export function buildPowerOn() {
  return new Uint8Array([0x7e, 0x00, 0x04, 0xf0, 0x00, 0x01, 0xff, 0x00, 0xef]);
}

export function buildPowerOff() {
  return new Uint8Array([0x7e, 0x00, 0x04, 0x00, 0x00, 0x00, 0xff, 0x00, 0xef]);
}

export function buildColor(r, g, b) {
  return new Uint8Array([0x7e, 0x00, 0x05, 0x03, clampByte(r), clampByte(g), clampByte(b), 0x00, 0xef]);
}

export function buildBrightness(percent) {
  return new Uint8Array([0x7e, 0x00, 0x01, clampPercent(percent), 0x00, 0x00, 0x00, 0x00, 0xef]);
}

// HSV → RGB. h in degrees [0, 360), s and v in [0, 1].
export function hsvToRgb(h, s, v) {
  const c = v * s;
  const hh = ((h % 360) + 360) % 360 / 60;
  const x = c * (1 - Math.abs((hh % 2) - 1));
  let r1 = 0, g1 = 0, b1 = 0;
  if (hh < 1)      { r1 = c; g1 = x; b1 = 0; }
  else if (hh < 2) { r1 = x; g1 = c; b1 = 0; }
  else if (hh < 3) { r1 = 0; g1 = c; b1 = x; }
  else if (hh < 4) { r1 = 0; g1 = x; b1 = c; }
  else if (hh < 5) { r1 = x; g1 = 0; b1 = c; }
  else             { r1 = c; g1 = 0; b1 = x; }
  const m = v - c;
  return {
    r: Math.round((r1 + m) * 255),
    g: Math.round((g1 + m) * 255),
    b: Math.round((b1 + m) * 255),
  };
}

// hue 0-360 → fully-saturated, full-value RGB
export function hueToRgb(hue) {
  return hsvToRgb(hue, 1, 1);
}

// (x, y) relative to wheel center, +x right, +y down (DOM coords).
// Returns hue in degrees [0, 360). 3 o'clock = 0°, going counter-clockwise like a real color wheel.
export function polarToHue(x, y) {
  // atan2 returns radians in (-π, π], with 0 at 3 o'clock and increasing counter-clockwise (positive y down inverts that).
  // Negate y so y-up matches mathematical convention.
  let deg = Math.atan2(-y, x) * 180 / Math.PI;
  if (deg < 0) deg += 360;
  return deg;
}

const KEY = 'light-hub:devices';
const CURRENT_SCHEMA = 1;

export class Store {
  constructor(storage = localStorage) {
    this.storage = storage;
    this._devices = this._load();
  }

  _load() {
    const raw = this.storage.getItem(KEY);
    if (!raw) return [];
    try {
      const parsed = JSON.parse(raw);
      if (parsed.schemaVersion !== CURRENT_SCHEMA) return [];
      return Array.isArray(parsed.devices) ? parsed.devices : [];
    } catch (e) {
      return [];
    }
  }

  _save() {
    this.storage.setItem(KEY, JSON.stringify({
      schemaVersion: CURRENT_SCHEMA,
      devices: this._devices,
    }));
  }

  list() {
    return [...this._devices];
  }

  add(partial) {
    const device = {
      localId: crypto.randomUUID(),
      bleId: partial.bleId || null,
      name: partial.name || 'Light',
      originalName: partial.originalName || partial.name || '',
      service: partial.service || null,
      writeChar: partial.writeChar || null,
      lastColor: partial.lastColor || { r: 255, g: 255, b: 255 },
      lastBrightness: typeof partial.lastBrightness === 'number' ? partial.lastBrightness : 100,
      lastOn: !!partial.lastOn,
      addedAt: new Date().toISOString(),
    };
    this._devices.push(device);
    this._save();
    return device;
  }

  findByLocalId(localId) {
    return this._devices.find(d => d.localId === localId);
  }

  findByBleId(bleId) {
    return this._devices.find(d => d.bleId === bleId);
  }

  findByOriginalName(name) {
    return this._devices.find(d => d.originalName === name);
  }

  rename(localId, newName) {
    const d = this.findByLocalId(localId);
    if (!d) return;
    d.name = newName;
    this._save();
  }

  remove(localId) {
    this._devices = this._devices.filter(d => d.localId !== localId);
    this._save();
  }

  updateState(localId, patch) {
    const d = this.findByLocalId(localId);
    if (!d) return;
    if (patch.lastColor !== undefined) d.lastColor = patch.lastColor;
    if (patch.lastBrightness !== undefined) d.lastBrightness = patch.lastBrightness;
    if (patch.lastOn !== undefined) d.lastOn = patch.lastOn;
    this._save();
  }

  updateBleId(localId, newBleId) {
    const d = this.findByLocalId(localId);
    if (!d) return;
    d.bleId = newBleId;
    this._save();
  }
}

const SERVICES = [
  '0000fff0-0000-1000-8000-00805f9b34fb',
  '0000ffe0-0000-1000-8000-00805f9b34fb',
];

const WRITE_CHAR_FOR_SERVICE = {
  '0000fff0-0000-1000-8000-00805f9b34fb': '0000fff3-0000-1000-8000-00805f9b34fb',
  '0000ffe0-0000-1000-8000-00805f9b34fb': '0000ffe1-0000-1000-8000-00805f9b34fb',
};

const NAME_PREFIXES = ['ELK-', 'MELK', 'LEDBLE', 'LED-', 'DAY', 'DB-'];

const REQUEST_FILTERS = {
  filters: [
    ...SERVICES.map(s => ({ services: [s] })),
    ...NAME_PREFIXES.map(p => ({ namePrefix: p })),
  ],
  optionalServices: SERVICES,
};

const CONNECT_TIMEOUT_MS = 8000;

function withTimeout(promise, ms, label) {
  return Promise.race([
    promise,
    new Promise((_, rej) => setTimeout(() => rej(new Error(`${label} timeout after ${ms}ms`)), ms)),
  ]);
}

export class ELKBLEDOMDriver {
  constructor() {
    this._chars = new Map();         // bleDeviceId → BluetoothRemoteGATTCharacteristic
    this._disconnectCbs = new Map(); // bleDeviceId → callback
  }

  isSupported() {
    return typeof navigator !== 'undefined' && !!navigator.bluetooth;
  }

  async requestDevice() {
    if (!this.isSupported()) throw new Error('Web Bluetooth not available');
    return await navigator.bluetooth.requestDevice(REQUEST_FILTERS);
  }

  async getKnownDevices() {
    if (!this.isSupported() || !navigator.bluetooth.getDevices) return [];
    try {
      return await navigator.bluetooth.getDevices();
    } catch (e) {
      return [];
    }
  }

  async connect(device, savedHints = {}) {
    const server = await withTimeout(device.gatt.connect(), CONNECT_TIMEOUT_MS, 'gatt.connect');

    const orderedServices = savedHints.service
      ? [savedHints.service, ...SERVICES.filter(s => s !== savedHints.service)]
      : SERVICES;

    let svc = null;
    let charUuid = null;
    for (const svcUuid of orderedServices) {
      try {
        svc = await server.getPrimaryService(svcUuid);
        charUuid = WRITE_CHAR_FOR_SERVICE[svcUuid];
        break;
      } catch (e) {
        // try next
      }
    }
    if (!svc) {
      device.gatt.disconnect();
      throw new Error('No supported service found on device');
    }

    const ch = await svc.getCharacteristic(charUuid);
    this._chars.set(device.id, ch);

    device.addEventListener('gattserverdisconnected', () => {
      this._chars.delete(device.id);
      const cb = this._disconnectCbs.get(device.id);
      if (cb) cb();
    });

    return { service: svc.uuid, writeChar: ch.uuid };
  }

  isConnected(device) {
    return this._chars.has(device.id);
  }

  async _write(device, bytes) {
    const ch = this._chars.get(device.id);
    if (!ch) throw new Error('Not connected');
    if (ch.properties.writeWithoutResponse) {
      await ch.writeValueWithoutResponse(bytes);
    } else {
      await ch.writeValue(bytes);
    }
  }

  async setPower(device, on) {
    await this._write(device, on ? buildPowerOn() : buildPowerOff());
  }

  async setColor(device, { r, g, b }) {
    await this._write(device, buildColor(r, g, b));
  }

  async setBrightness(device, percent) {
    await this._write(device, buildBrightness(percent));
  }

  disconnect(device) {
    try { device.gatt.disconnect(); } catch (e) { /* ignore */ }
    this._chars.delete(device.id);
  }

  onDisconnect(device, callback) {
    this._disconnectCbs.set(device.id, callback);
  }
}

// Hash router. Routes:
//   #/                  → { name: 'list' }
//   #/light/<localId>   → { name: 'detail', localId }

export function parseRoute(hash = location.hash) {
  const path = (hash || '').replace(/^#/, '');
  if (!path || path === '/') return { name: 'list' };
  const m = path.match(/^\/light\/([^/]+)$/);
  if (m) return { name: 'detail', localId: m[1] };
  return { name: 'list' }; // unknown route → home
}

export function navigate(routeName, params = {}) {
  if (routeName === 'list') {
    location.hash = '#/';
  } else if (routeName === 'detail') {
    location.hash = `#/light/${params.localId}`;
  }
}

// Back-button safe: if no history, go to list instead of exiting.
export function back() {
  if (history.length > 1) {
    history.back();
  } else {
    navigate('list');
  }
}

export function onRouteChange(callback) {
  window.addEventListener('hashchange', () => callback(parseRoute()));
}

export function placeWheelPin(pinEl, wheelEl, hue) {
  const rect = wheelEl.getBoundingClientRect();
  const radius = rect.width * 0.42;
  // Inverse of polarToHue: hue → angle in standard math (counter-clockwise from 3 o'clock)
  // and convert back to DOM coords (y inverted).
  const rad = hue * Math.PI / 180;
  const x = Math.cos(rad) * radius;
  const y = -Math.sin(rad) * radius;
  pinEl.style.left = `${rect.width / 2 + x}px`;
  pinEl.style.top = `${rect.height / 2 + y}px`;
}

export function hueFromPointer(wheelEl, clientX, clientY) {
  const rect = wheelEl.getBoundingClientRect();
  const cx = rect.left + rect.width / 2;
  const cy = rect.top + rect.height / 2;
  return polarToHue(clientX - cx, clientY - cy);
}

function rgbToHue(r, g, b) {
  const rn = r / 255, gn = g / 255, bn = b / 255;
  const max = Math.max(rn, gn, bn), min = Math.min(rn, gn, bn);
  const d = max - min;
  if (d === 0) return 0;
  let h;
  if (max === rn) h = ((gn - bn) / d) % 6;
  else if (max === gn) h = (bn - rn) / d + 2;
  else h = (rn - gn) / d + 4;
  h = h * 60;
  if (h < 0) h += 360;
  return h;
}

// Pair flow

async function pairFlow() {
  let device;
  try {
    device = await driver.requestDevice();
  } catch (e) {
    return; // user cancelled or no devices
  }
  // Resolve identity
  let record = store.findByBleId(device.id) || store.findByOriginalName(device.name);
  if (record) {
    if (record.bleId !== device.id) {
      const ok = confirm(`Is this "${record.name}"?`);
      if (ok) store.updateBleId(record.localId, device.id);
      else record = null;
    }
  }
  // Connect
  let connInfo;
  try {
    setStatus(record?.localId || '__pending__', 'CONNECTING');
    connInfo = await driver.connect(device, record || {});
  } catch (e) {
    alert(`Couldn't connect: ${e.message}`);
    if (record) setStatus(record.localId, 'OFFLINE');
    return;
  }
  // Add new record if needed
  if (!record) {
    const name = await promptName(device.name);
    if (!name) { driver.disconnect(device); return; }
    record = store.add({
      bleId: device.id,
      name,
      originalName: device.name,
      service: connInfo.service,
      writeChar: connInfo.writeChar,
    });
  }
  setDeviceRef(record.localId, device);
  setStatus(record.localId, 'CONNECTED');
  driver.onDisconnect(device, () => handleDisconnect(record.localId));
  render();
}

function promptName(suggested) {
  return new Promise((resolve) => {

    document.body.appendChild(wrap);
    const cleanup = () => wrap.remove();
    wrap.addEventListener('click', (e) => {
      const action = e.target.dataset?.action;
      if (action === 'modal-cancel') { cleanup(); resolve(null); }
      else if (action === 'modal-save') {
        const v = document.getElementById('modal-name-input').value.trim();
        cleanup();
        resolve(v || suggested);
      }
    });
  });
}

async function connectAll() {
  for (const record of store.list()) {
    if (statusOf(record.localId) === 'CONNECTED') continue;
    try {
      const device = await driver.requestDevice();
      // user picks; if it's a different one we still try to bind
      let target = store.findByBleId(device.id) || store.findByOriginalName(device.name);
      if (!target) target = record; // fall back: associate with current iteration's record
      setStatus(target.localId, 'CONNECTING');
      const info = await driver.connect(device, target);
      store.updateBleId(target.localId, device.id);
      setDeviceRef(target.localId, device);
      setStatus(target.localId, 'CONNECTED');
      driver.onDisconnect(device, () => handleDisconnect(target.localId));
    } catch (e) {
      // User cancelled or connect failed; stop the chain
      break;
    }
  }
  render();
}

async function reconnect(localId) {
  const record = store.findByLocalId(localId);
  if (!record) return;
  setStatus(localId, 'CONNECTING');
  try {
    const known = await driver.getKnownDevices();
    let device = known.find(d => d.id === record.bleId);
    if (!device) device = await driver.requestDevice();
    const info = await driver.connect(device, record);
    if (device.id !== record.bleId) store.updateBleId(localId, device.id);
    setDeviceRef(localId, device);
    setStatus(localId, 'CONNECTED');
    driver.onDisconnect(device, () => handleDisconnect(localId));
  } catch (e) {
    setStatus(localId, 'OFFLINE');
  }
}

  await reconnect(localId);
}

    if (statusOf(localId) !== 'DISCONNECTED') return;
    await reconnect(localId);
  }, 500);
}

async function autoReconnect() {
  if (!driver.isSupported()) return;
  const known = await driver.getKnownDevices();
  if (known.length === 0) return; // either no permission persistence or first launch
  for (const record of store.list()) {
    const device = known.find(d => d.id === record.bleId);
    if (!device) continue;
    setStatus(record.localId, 'CONNECTING');
    try {
      await driver.connect(device, record);
      setDeviceRef(record.localId, device);
      setStatus(record.localId, 'CONNECTED');
      driver.onDisconnect(device, () => handleDisconnect(record.localId));
    } catch (e) {
      setStatus(record.localId, 'OFFLINE');
    }
  }
}

# light-hub

A web-based controller for ELK-BLEDOM-family BLE LED strips (Lotus Lantern, DayBetter, and many cheap Amazon LED strips that use the same protocol). Runs in Bluefy on iOS or Chrome on desktop / Android. No native app, no Apple Developer account, no hub.

- **Desktop Chrome / Edge:** open the URL directly. Useful for testing.

## Add a light

## Use

## Known limitations

- **Web Bluetooth quirks:** if `getDevices()` isn't supported in your Bluefy version, you'll need to tap each tile's "Connect" once per session.
- **White quality on RGB-only strips:** the W channel of RGBW strips isn't used in this version. White via RGB(255,255,255) may look bluish.
- **Two phones, one strip:** BLE allows only one connection per device. Second phone will see "couldn't connect" — disconnect from the first phone first.
- **localStorage clearing:** if Bluefy clears site data, you'll re-pair all lights.

## Troubleshooting

**My strip doesn't appear in the picker.** Make sure it's powered on, in BLE range (~10m, no walls), and not currently connected to another phone or app. Force-quit any vendor app (DayBetter, Lotus Lantern) before pairing here.

**"Couldn't connect."** Try once more. If it persists, restart the strip (unplug/replug). If still failing, the strip's protocol may differ — open `probe.html` and run the buttons to diagnose.

**Color wheel doesn't move.** Make sure the strip status is green (CONNECTED). If yellow or red, tap the offline banner to reconnect.
